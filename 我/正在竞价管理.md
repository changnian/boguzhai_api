###获取正在竞价的拍品列表
Http Request: 

```
m=getBiddingLotList
sessionid=1h283019400031n22hhu22313145252
```

Http Response:

- code=0 : 获取信息成功
- code=-1 : sessionid 错误
- code=1 : 由于其它原因，获取信息失败
