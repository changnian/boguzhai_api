### <a name="1">获取拍品分类映射表</a>

Http Request: 

```
m=getAuctionTypeMap
```
Http Response:
 
- code=0 : 获取信息成功
- code=1 : 获取信息失败，服务器错误

### <a name="2">获取省市区映射表</a>

Http Request: 

```
m=getAddressZoneMap
```
Http Response:
 
- code=0 : 获取信息成功
- code=1 : 获取信息失败，服务器错误

注：省/直辖市/特别行政区 -> 市 -> 区/县/市


``` 
{ 
    "code": 0;
    "data": {
    	"addressZoneMap":{
    		"江苏": {
    			"南京": ["玄武", "鼓楼", "仙林", ..., "江宁"];
    			... ;    			
    			"苏州": ["平江", "沧浪", ..., "苏州工业园"];
    		};
    		"浙江": {
    			"杭州": ["上城", "下城", "西湖", ..., "萧山"];
    			... ;    			
    			"宁波": [ ... ];
    		};
    		... ;
    		... ;	
    		"上海": {
    			"上海": ["黄浦", "虹口", ..., "浦东新"];
    		};
    	}
    }
}
```
