###按条件查询拍品

Http Requset : 

```
m=searchAuction
keyword=keyword #关键字
lotType1=古籍书刊 #拍品一级分类
lotType2=线状刻本 #拍品二级分类
lotType3=明代刻本 #拍品三级分类
auctionId=AC1001212 #拍卖会编号
auctionPartId=3 #拍卖会分场编号
bidType=2 #拍品拍卖形式
lotStatus=1 #拍品状态
dealType=0 #成交方式
```

Http Response : 

- code=0 : 获取信息成功，返回拍品列表
- code=1 : 获取信息失败，服务器错误

``` 
{ 
    "code": 0;
    "data": {
    	"auctionIdList":[ "231974", "213724", "981904", ..., "814981"]
	}
}
```
