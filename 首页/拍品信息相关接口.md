###<a name="1">获取首页拍品列表</a>
Http Request: 

```
m=getMainLotList
```
Http Response:

- code=0 : 获取信息成功，返回拍品ID列表
- code=1 : 获取信息失败，服务器错误

``` 
{ 
    "code": 0;
    "data": {
    	"lotIdList":[ "231974", "213724", "981904", ..., "814981"]
	}
}
```
