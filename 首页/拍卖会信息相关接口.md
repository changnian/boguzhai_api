### <a name="1">获取首页拍卖会专场列表</a>
**测试接口** <>

Http Request: 

```
m=getMainAuctionSessionList
```
Http Response:

- code=0 : 获取信息成功，返回拍卖会专场ID列表
- code=1 : 获取信息失败，服务器错误

``` 
{ 
    "code": 0;
    "data": {
    	"auctionSessionIdList":["2314","2137","9814",...,"8181"]
    }
}
```
